# Guess The Number - Gra Konsolowa C++

Prosta gra tekstowa polegająca na odgadnięciu wylosowanej liczby przez komputer.

## Funkcje
- **3 Poziomy trudności**: Łatwy (1-50), Średni (1-100), Trudny (1-250).
- **System TOP5**: Zapisuje najlepsze wyniki (najmniejsza liczba prób) do pliku tekstowego.
- **Tryb Zakładu**: Możliwość ustawienia własnego limitu prób.
- **Interfejs**: Czytelne nagłówki i dynamiczne komunikaty.

## Instrukcja Uruchomienia
1. Upewnij się, że masz zainstalowany kompilator C++ (np. g++).
2. Skompiluj program:
   g++ -o gra_zgadnij main.cpp

## Sterowanie
- Poruszanie się po menu odbywa się poprzez wpisanie odpowiedniej cyfry i naciśnięcie Enter.
- W trakcie gry wpisuj liczby całkowite, aby zgadnąć wynik.
- Po wygranej wpisz swoje imię (jeden wyraz), aby zapisać się w rankingu.
